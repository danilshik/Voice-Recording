# Приложение для записи звука Voice-Recording [Еще разрабатывается]
Паттерн : MVVM.

Язык программирования : Kotlin

Используемые компоненты:
+ Dagger2
+ Room
+ LiveData
+ ViewModel
+ Coroutine
+ Service
+ BroadcastReceiver
+ Различные элементы из Material Design

В будущем возможно добавиться:
+ DataBinding
+ Paging Library
